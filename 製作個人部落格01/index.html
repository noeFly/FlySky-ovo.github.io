教學動機

其實你所看到的目前這個網站就是用 Hexo 寫出來的，準確一點，應該是「生成」出來的。
我是個喜歡自己搞東西的人，如果可以的話會自己弄一個而非直接用網路上的資源來弄。
應該就差不多時這樣，嗯(?

學習難度

這不難，主要編寫的時候是寫 MarkDown 而不是 Html，也不用用手搓出來一個 CSS 或是 JavaScript。
這些麻煩的東西基本上都會有主題作者編寫完成，我們只要調整一下設定就好了！

為什麼要做網站？

嗯…可以凸顯你的文學氣息
（請忽略上面那句話）

做個網站可以讓你放一下你想要的文章、教學文、自創小說、日記甚至是自己做的東西介紹都可以放在這裡。
而且多寫一些文章還有益提升國文能力！（這時候我在準備會考）

大概就是這樣了，你我們就下一篇文章見嘍！

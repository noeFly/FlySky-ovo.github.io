Visual Studio Code

這是我們在編輯設定、編輯文章時使用的文字編輯器。
並且支援安裝擴充套件，讓我們編輯部落格的時候更加方便！

安裝

先到 VS Code 的官網
有一個大大藍藍的按鈕給他點下去就好了！

Node.js

簡單一點就是整個部落格的核心，他會負責處理大部分的事情。

安裝

  1. 首先前往 Node.js 官網

  2. 選擇「長期維護版（LTS）」

    在官網你會發現有兩個版本，分別是長期維護版（LTS）和最新版本（Current）

    長期維護版（LTS）會比較穩定沒有那麼多的問題

    最新版本（Current）則是會有新的功能，可以嘗鮮一下，但是相對的就沒那麼穩定。

  3. 點擊左邊的長期維護版，下載後照著步驟做，Node.js 就弄好嘍！

Hexo

Hexo 是我們部落格的核心套件，負責幫我們生成網站的檔案、處理各種漂亮主題等。

安裝

下一篇就會正式開始編輯網站，加油！

創建專案

創建資料夾

首先我們要先找一個地方放我們的網站檔案
像我自己就是把檔案放在 D: 裡面
也可以放在桌面 看你個人喜好！
創建一個資料夾，名稱隨便，盡量避免中文字！

開啟 Visual Studio Code

打開資料夾後右鍵以 Code 開啟

打開終端機

使用快捷鍵 Ctrl + `開啟終端機
預設應該會在下面開啟！

Node.js 初始化

在剛剛開啟的終端機裡面輸入以下指令

接著他會問一連串的問題

package name

名稱：你的這個專案要叫什麼名字

version

版本：這個其實沒啥用，輸入1.0.0就好了

description

簡介：簡單介紹一下你的這個專案是做什麼用的。

entry point

進入點：這個不要動直接Enter就好了！

test command

除錯指令：Enter 跳過～

git repoisty

Git 專案：這個我們之後會調整，Enter跳過～

keyword

關鍵字：應該是 SEO 用的，Enter跳過～

author

作者：就是你叫什麼名字

license

授權：你希望以什麼樣的方式公開你的專案

Is this ok?

你確定嗎？：這是確認訊息，確認你的資料沒有打錯後輸入yes

完成後你應該會看到你的專案資料夾裡面多了一個package.json
裡面就是剛剛輸入的資料全部都在這裡！

恭喜你已經創建好專案了！
